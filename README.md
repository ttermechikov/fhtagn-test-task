# Выполнение [тестового задания компании Фхтагн](https://docs.google.com/document/d/1kMcmgg9GkUu9wcDjfyZC3MsJW2G21lzHCuzM8iTPsK0/edit)

## Установка зависимостей

```
npm install
```

### Запустить режим разработки

```
npm run serve
```

### Сборка для деплоймента

```
npm run build
```

### Линт

```
npm run lint
```
